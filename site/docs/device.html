The Device Tab

The Device tab is where you adjust the scanner hardware and set a couple of preferences for the app. It's split across six pages, one per section.

Changes to scanner settings apply immediately. There's no Save button. When you move a slider or flip a switch on any of the hardware pages, the app enters the scanner's programming mode to write the change, and live scanning pauses while that write happens. A failed write shows a red error message. (Preferences is the exception — it's app-side only and never touches the radio.)
