What it does

Live display

Readable across the room

Frequency, alpha tag, modulation, and signal strength, updated live while the scanner runs. No squinting at a two-line LCD.

Channel memory

Edit all 500 channels

Every channel, bank, tone, and lockout — edited from a real keyboard instead of the radio's keypad.

Import / export

Plain CSV in and out

Move channel lists in and out of the radio as CSV or native Uniden files. Your data stays yours.

Activity logging

Every hit, on the record

The one thing the radio and the official software never did: Bearpaw logs every hit while it runs and hands you the raw data as CSV whenever you want it.

Wire protocol

Speaks fluent Uniden

Talks to the scanner the same way Uniden's Sentinel software does, over the USB cable — verified against real wire captures.

Cross-platform

macOS, Windows, Linux

One app, built with Tauri, running live while you listen — on whatever machine sits next to the radio.

    This is a beta. It's been tested against real hardware, but you may hit rough edges. Found one? Open an issue.

    Beta builds aren't code-signed yet, so your computer flags them the first time you open the app:

    • macOS: right-click the app and choose Open, then Open again. Still refuses? Run xattr -cr /Applications/Bearpaw.app once in Terminal.
    • Windows: at “Windows protected your PC,” click More info, then Run anyway.
    • Linux: chmod +x the AppImage and run it, or install the .deb.
